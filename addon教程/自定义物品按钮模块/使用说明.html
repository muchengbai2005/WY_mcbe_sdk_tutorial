功能描述

主要功能：为物品添加hud界面的按钮

  • 可为任意物品添加屏幕上的按钮
  • 支持设置按钮贴图，回调函数，按钮位置等
  • 为多个物品添加按钮，不受数量限制

使用方法

  • 将AetherPresetButtonUI.json放在资源包ui文件夹里，并配置在_ui_defs.json。

  • 将AetherPresetButton文件夹放在行为包里。

  • 在UiInitFinished事件后调用接口注册物品按钮：

    @Listen
def UiInitFinished(self, args):
    presetButton = clientApi.GetSystem('AetherPresetButton', 'ClientSystem')
    # self.CreateMainUI换成你自己的回调函数，后两个参数可选。
    presetButton.SetItemButton('minecraft:stick', self.CreateMainUI, '使用', '右键使用')

模块接口

SetItemButton

  • 描述

    注册物品按钮

  • 参数

    参数名 数据类型 说明
    itemName str 物品的identifier，即'命名空间:物品名'
    callBack function 按钮释放后触发的函数，即你想让此按钮触发的逻辑
    buttonText str 可选，按钮上的文字，默认为''
    pcTips str 可选，非触屏模式文字提示，默认为''
    default str 可选，按钮的default贴图，默认为原版按钮贴图
    hover str 可选，按钮的hover贴图，默认为原版按钮贴图
    pressed str 可选，按钮的pressed贴图，默认为原版按钮贴图
    size tuple 可选，按钮的尺寸，默认为(30, 18)
    anchorFrom str 可选，按钮的父锚点，默认为'bottom_middle'
    anchorTo str 可选，按钮的锚点，默认为'bottom_middle'
    offset tuple 可选，按钮相对于锚点的偏移量，默认为(0, -43)

  • 示例

    import client.extraClientApi as clientApi
presetButton = clientApi.GetSystem('AetherPresetButton', 'ClientSystem')
presetButton.SetItemButton('minecraft:stick', self.CreateMainUI, '使用', '右键使用')

  • 备注

    一定要在UiInitFinished事件后再调用本接口，不要写在__init__里。
